🚀 Deploy WBIntel
GITHUB → NETLIFY AUTO-DEPLOY SETUP GUIDE
✅ Total cost: ₹0/month forever · Auto-deploy on every push · Free SSL + CDN
Step 1 — Create GitHub Repository
1
Go to github.com → Sign in (or create free account) → Click + New repository
2
Name it wbintel → Set to Public → Click Create repository
3
On your computer: Download this index.html file → Open the repository page → Click Add file → Upload files
4
Drag and drop index.html into GitHub → Write commit message initial deploy → Click Commit changes
Step 2 — Connect Netlify (Auto-Deploy)
1
Go to app.netlify.com → Sign up free with your GitHub account
2
Click Add new site → Import an existing project → GitHub
3
Select your wbintel repository → Leave all build settings blank (it's plain HTML) → Click Deploy site
4
In ~30 seconds your site is live at random-name.netlify.app → Go to Site settings → Domain → Change site name → rename to wbintel.netlify.app
Step 3 — Auto-Deploy (Push = Publish)
✅
Done! Now whenever you update index.html on GitHub, Netlify automatically re-deploys in ~15 seconds. Zero manual steps.
→
To update: GitHub → your repo → click index.html → pencil icon → edit → Commit changes → Netlify auto-updates live
Step 4 — Custom Domain (Optional)
1
Free domain: Register wbintel.rf.gd or wbintel.kesug.com at infinityfree.net
2
Paid .in domain (~₹499/yr): Register at GoDaddy/BigRock → Netlify site settings → Add custom domain → add DNS records
🔗 Your live site URL example: https://wbintel.netlify.app

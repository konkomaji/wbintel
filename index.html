ℹ About WBIntel

WBIntel

West Bengal State Intelligence Monitor v6.0

A real-time intelligence dashboard aggregating news, social media, weather, live TV, and district-level activity across all 23 districts of West Bengal.

Built for journalists, researchers, political analysts, and civic watchers who need total situational awareness.

Created by Konko Maji · MIT License · Open Source

All data fetched live from public APIs & RSS feeds. No backend required.

🚀 Deploy Guide

GitHub Setup

  1. Go to github.com → New repository → name wbintel → Public → Create
  2. Upload all project files via Add file → Upload files
  3. Commit with message initial deploy

Netlify Auto-Deploy

  1. Go to app.netlify.com → Sign up with GitHub → Add new site → Import from GitHub
  2. Select wbintel repo → Leave build settings empty (plain HTML) → Deploy
  3. Rename site to wbintel.netlify.app in Site settings
  4. Done — every push to GitHub auto-deploys in ~15 seconds
